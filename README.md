# ImageRecognition

1. Get target data (images to recognize)
1-1. Run “main.py”
1-2. Press “p” to take photos (about 10 pcs)
1-3. Confirm that photos are stored in “pictures” folder
1-4. Press “q” to stop running

2. Data processing and Machine Learning (Data preparation and ML model build)
2-1. Run “DataLoad_and_Learn.py”
2-2. Wait for about 10 min
2-3. Confirm that below three files are stored in “model” folder
	model.json
	train.csv
	weights.h5

3. Abnormality detection
3-1. Run “main.py”
3-2. Put a sample in camera frame, and see score.
